  • Presenter: Shaoquan Jiang
  • Topic: A short course by Dr. Shaoquan Jiang: "Security of cryptographic protocols"
    Description: In this short course, a set of examples, such as key exchange, broadcast encryption, and traitor tracing, will be used to show how the security of a cryptographic protocol can be formally treated. This course will also cover how a sequence of game techniques can be used to prove the security of a well designed protocol. Some cryptographic notions such as encryption (semantic security, CCA1 security and CCA2 security), pseudorandom function, message authentication code will be introduced when necessary. Note that since we do not have much time, it is impossible to systematically introduce the complete theory of the cryptographic protocol. Instead, we first give introduction to the security of protocol in the intuitive sense. Then we introduce ideal-real emulation based model. Then we study some security treatment in a few papers.
