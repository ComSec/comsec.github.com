GH-DSA: Signing
•System Parameters:
–f(x) = x3 – ax2 + bx – 1, an irreducible polynomial over GF(p),
–where p is a prime number.  Period is Q is a factor of p2 + p + 1.
•
•Signing process:
1.Compute public key pair (Algorithm 1).
2.Randomly choose k, with 0 < k < Q and gcd(k,Q) = 1.  Compute sk (Algorithm 1).  Set r = sk if gcd(sk,Q) = 1.
3.Compute h = h(m), where m is the message to be signed and h(·) is a hash function.
4.Compute t = k-1(h - rx) mod Q.
•
•(r, t) is a digital signature of the message m.  Alice sends (m,r,t), (sx, sx+1), (sk, sk+1) and their duals to Bob.
